# Jammming

## Description
Jammming is a Codecademy portfolio project and my attempt to build a React web app that integrates with the Spotify API.

The goal of the app is to let users:
- Search the Spotify library for songs
- Create custom playlists
- Save those playlists directly to their Spotify account

This project is focused on practicing real-world frontend development with React, API integration, and user authentication/authorization flows.
